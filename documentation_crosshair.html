Crosshair Color (cg_crosshairColor)

SettingDescriptionTags
Setting: [color]Description:

Entering a single color by itself means that the crosshair uses this color at all times. Use any color name or hex code, e.g. purple or #ff00ff.

Note that crosshair mods containing colors in the file itself (e.g. "red dot mod") will not properly support Newmod's coloring system. Crosshair mods must be white for custom colors to work properly.

Setting: idle=[color]Description: Crosshair uses this color when idle.Tags: Optional Optional
Setting: ally=[color]Description: Crosshair uses this color when aiming at allies.Tags: Optional Optional
Setting: enemy=[color]Description: Crosshair uses this color when aiming at enemies.Tags: Optional Optional
Setting: neutral=[color]Description: Crosshair uses this color when aiming at neutral entities (breakables, NPCs, etc).Tags: Optional Optional
Setting: idle=white ally=lime neutral=yellow enemy=redDescription: Default setting.Tags: Default Default Base Base
