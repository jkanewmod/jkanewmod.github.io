New in version 1.12.0!
For NewJK users only.
SettingDescriptionTags
Setting: 0Description: Disable.Tags: Default Default Base Base
Setting: 1Description: Prevent JKA from losing focus or being minimized during live matches.

This setting is useful if random background programs sometimes cause your game to minimize.

"Live match" means you and at least 3 others are ingame, and on supporting servers: the match was restarted, nobody was AFK, and the match isn't paused.

Auto Download Fake Map (cg_fakeMap)

SettingDescriptionTags
Setting: 0Description: Disable.Tags: Default Default Base Base

Auto Switch Upon Pickup (cg_autoSwitch)

Updated in version 1.12.0!
Base JKA cvar enhanced by Newmod.
Newmod's implementation of this base JKA cvar fixes many of base JKA cg_autoSwitch's bugs and inconsistencies. It is now split into two different modes, which both avoid switching in certain scenarios, such as while equipping saber while carrying the flag, or while equipping mines/detpacks.
See also: Weapon Preference (cg_weaponPreference).
SettingDescriptionTags
Setting: 0Description: Disable.

Most experienced players use this setting.
Setting: 1Description: Enable simple mode: just save the hassle of manually switching to better weapons upon pickup.

This has always been the default setting in JKA, despite the fact that most experienced players disable it.		Tags: Default Default
Setting: 2Description: Enable babysitter mode: have the game aggressively switch weapons for you in more scenarios.

Force Game (fs_forceGame)

For NewJK users only.
This cvar can only be set via command line arguments. It cannot be set after JKA has already been started. You must add +set fs_forceGame "[mod name]" to your command line arguments to use it.
SettingDescriptionTags
Setting: 0 or ""Description: Disable. Allows servers to freely change your fs_game setting.Tags: Default Default Base Base
Setting: [mod name]Description: Force the fs_game cvar to [mod name]. Annoying servers will be prevented from changing your game mod directory. Use base to force it to an empty "" value (for base directory).
