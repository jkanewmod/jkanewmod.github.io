Allow Minimize (cl_allowMinimize)

For vanilla executable users only

SettingDescriptionTags
Setting: 0Description: Disable.Tags: Base Base
Setting: 1Description: On vanilla executable, force JA to minimize whenever it loses focus + automatically change screen resolution.Tags: Default Default
Setting: 2Description: On vanilla executable, force JA to minimize whenever it loses focus.

Automatic Downloading (cg_autoDL)

SettingDescriptionTags
Setting: 0Description: Disable.Tags: Base Base
Setting: 1Description: Automatically download missing maps over HTTP connection (requires compatible server).Tags: Default Default

Auto Download Fake Map (cg_fakeMap)

SettingDescriptionTags
Setting: 0Description: Disable.Tags: Default Default Base Base
Setting: [map name]Description: If you don't have a map and autoDL fails (due to no autoDL server or map missing from the autoDL server), you can specify a map to load instead of the real map. It will probably look buggy, but you will at least be able to connect to the server (and thereby vote for a new map, etc). For best results, choose a map similar to the real map (e.g. choose a CTF map for CTF).

Chat URLs (cg_chatURLs)

SettingDescriptionTags
Setting: 0Description: Disable detection of URLs in chat.Tags: Base Base
Setting: 1Description: After someone writes a URL in chat, it is assigned an ID number. You can then use open [id] to open the URL in your default browser, or copy [id] to copy it to the clipboard (you may optionally leave out [id] to simply open/copy the last detected URL).Tags: Default Default

Setting: mainSize=[size]Description: Size of primary fire shots. Uses base tapering from 6 to 0 if unspecified; set to something else for constant size.Tags: Optional Optional
Setting: altSize=[size]Description: Size of alternate fire shots. Uses base tapering from 6 to 0 if unspecified; set to something else for constant size.Tags: Optional Optional

Force Models (cg_forceModel)

Base cvar enhanced by Newmod

SettingDescriptionTags
Setting: 0Description: Disable.Tags: Default Default
Setting: ally=[model]Description: Force allies to use [model] player model in non-Siege gametypes.Tags: Optional Optional
Setting: enemy=[model]Description: Force enemies to use [model] player model in non-Siege gametypes.Tags: Optional Optional

Note for advanced users: if you change forced models using auto CFGs, your forced models might not be automatically precached if they are dependent on an auto CFG executing first. To solve this, use the command nm_cacheSkin to force your desired models to be loaded into memory.
