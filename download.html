Download the latest version of the NewJK executable for Windows (December 20, 2019).

NewJK is our optional improved custom executable that replaces jamp.exe, which launches when you start JKA. It can improve FPS, has many crash fixes and stability improvements, and adds extra capabilities like raw mouse input. While NewJK is not required to use Newmod, we highly recommend it. For more information, see the NewJK page.

If you are also already using Newmod, make sure to update that, too.

To install NewJK manually, put jamp.exe, rd-vanilla_x86.dll, and SDL2.dll in the folder where your existing jamp.exe is located (this is usually the /GameData/ folder). We recommend backing up your existing jamp.exe first.

Release notes:


  • Chat translation

    • You can now automatically translate chats. Set a player to be translated with the Player Tracker command (nm_playerTracker) using the format: nm_playerTracker translate <client number> <their language>.

    • Set your own native/preferred language with the Translate To cvar (cg_translateTo), which defaults to english.

  • Vchat

    • You can now send custom (non-base JKA) voice chats to other players with the new Vchat command (nm_vchat). Modders can easily create vchat packs compatible with Newmod, which you can send and receive on compatible servers (none are included with Newmod).
    • You can choose which categories of vchats to enable with the Vchat cvars. Vchats are one of three categories: teamwork, general, or meme.
    • You can mute vchats from a particular player with the Player Tracker command (nm_playerTracker) using the format: nm_playerTracker mute <client number>.

  • Key binds

    • NewJK executable only: You can now bind key combinations using an Alt / Ctrl / Shift modifier key. For example, bind ctrl+x kill will make Ctrl + X perform kill.
    • NewJK executable only: You can now bind the right-side Alt / Ctrl / Shift keys separately from the left-side keys. For example, bind rctrl kill will make the right-side Ctrl key perform kill. Don't worry, your existing configuration is still 100% compatible — if no binding is specifically set for the right-side key, then the generic binding is used. The left-side key always uses the generic binding.

  • Miscellaneous new features

    • Added the new Sky Portals cvar (cg_skyPortals), which controls whether to render sky portals (distant mountains/buildings, etc.). Disabling this cvar may improve FPS on some maps.
    • Added the new FPS Test command (nm_fps), which measures average FPS over time.
    • Added the new Mute command (mute), which toggles playback of all in-game sound.
    • Added the new Chat Beep cvar (cg_beep), which controls when to play the chat beep sound.
    • Added the new Fast Sky Color cvar (r_fastSkyColor), which changes the color of the sky when the base JKA Fast Sky cvar (r_fastSky) is used.
    • Added the new Language Fixes cvar (cg_languageFix), which applies fixes for certain languages (French, Hungarian).
    • NewJK executable only: added the new Force Game cvar (fs_forceGame), which forces the fs_game cvar to a certain setting, preventing annoying servers from changing your game mod folder.
    • The Gun Display cvar (cg_drawGun) now supports decimal numbers between 0 and 1 to display your weapon with partial opacity.

  • Miscellaneous improvements

    • You can now type the Euro symbol € in the chat. This replaces the useless/troll "WSI FONTS" character.
    • You can now see scoreboard data any time while watching a demo; you don't need to wait until the person recording the demo opened their scoreboard or died.
    • Added improved support for CTF race mode on base_enhanced servers.
    • The strafe helper speedometer (cg_strafeHelper) now also shows your speed from the last time you were on the ground.
    • The FPS Saver cvar (cl_fpsSaver)'s 2 bitflag now saves even more FPS.
    • The Siege Help cvar (cg_drawSiegeHelp) has been enhanced to display floating 3D messages in the game world (requires server support).
    • While connected to a base_entranced siege server, siege class information displayed in the "Join" menu now displays accurate data sent from the server.
    • Numerous other miscellaneous improvements.

  • Miscellaneous bug fixes

    • NewJK executable only: fixed mouse getting stuck in the center of your screen when minimizing JKA immediately after startup.
    • NewJK executable only: fixed pressing AltGr and another key causing certain behaviors erroneously, e.g. AltGr + V performing paste.
    • Fixed CTF flag capture HUD notifications from the Capture Display cvar (cg_drawCaptures) using the wrong color. Correct colors are now used (e.g. when red team scores, red color is used).
    • Fixed a base JKA behavior where your own vchat sounds would be stopped by jumping or rolling, despite other players being able to still hear it.
    • Fixed the "phantom saber hum" bug, where you could occasionally hear saber hum sounds from dead players.
    • Fixed the removeOnDeath parameter of the Countdown Timer command (nm_timer) not working.
    • Numerous other miscellaneous bug fixes.

  • New features for power users

    • Added the new Random Command command (random), which executes a random command from a list.
    • Added the new String Substitution command (strSub), which executes a command, substituting in the contents of cvar(s).
    • Added the new If Cvar command (ifCvar), which executes a certain command depending on the setting of a cvar.
    • The Shell Command command (shell) has been updated to set a cvar to your command's output.

  • 1.5.1 hotfix notes

    The 1.5.1. hotfix release includes a few QoL improvements from 1.5.0.

    • Disabled and muted vchats are now 100% silent and hidden (previously, their sound was simply changed to a beep).

    • The default setting of the Meme Vchats cvar (cg_vchatMeme) is now 3 instead of 2. This means that by default, you will no longer see non-installed meme vchats whatsoever.
