Download the latest version of the NewJK executable for Windows (July 23, 2021).

NewJK is our optional improved custom executable that replaces jamp.exe, which launches when you start JKA. As a custom executable, it is capable of implementing many improvements that are not possible with simple PK3 mods (like Newmod). It can improve FPS, has many crash fixes and stability improvements, and adds extra capabilities like raw mouse input. While NewJK is not required to use Newmod, we highly recommend it. For more information, see the NewJK page.

If you are also already using Newmod, make sure to update that, too.

To install NewJK manually, put jamp.exe, rd-vanilla_x86.dll, SDL2.dll, steam_api.dll, and steam_appid.txt in the folder where your existing jamp.exe is located (this is usually the /GameData/ folder). We recommend backing up your existing jamp.exe first.

Release notes for 1.7.0:




  • Dropped flags and weapons

    • Added a new Dropped Flags cvar (cg_droppedFlags), which customizes the appearance of dropped flags. Defaults to base JKA behavior.
    • Added a new Dropped Weapons cvar (cg_droppedWeapons), which customizes the appearance of dropped weapons. Defaults to base JKA behavior.

  • Sounds

    • Added a new Ambient Sounds cvar (cg_ambientSounds), which toggles useless ambient sounds. Defaults to base JKA behavior.
    • Added a new Match Start Spam cvar (cg_matchStartSpam), which toggles sound/effects spam on match start. Defaults to base JKA behavior.
    • Vchats from spectators now use the voice specific to their chosen character model.
    • Improved how sounds are handled while minimized: sounds are now allowed to freely start, progress, and finish while minimized; they are simply inaudible (previously, sounds would pause while minimized and resume when restored).
    • Fixed sounds bugging out and looping during "Server is changing maps".

  • App integrations

    • Added a new Discord Integration cvar (cl_discord), which allows your current server info to be displayed in your Discord status. Defaults to enabled (only does anything if you have game activity enabled in Discord).
    • Added a new Steam Integration cvar (cl_steam), which allows starting NewJK directly to enable the Steam overlay. Defaults to disabled.

  • Commands/scripting

    • Nested quotation marks are now supported in commands and binds, allowing for easier and more robust configuration. This allows entering commands like bind a "bind b "set c "d ; e" ; say "hello there"""
    • Added new Delay/Wait Freely commands (delay, waitf), which are enhanced versions of the base JKA wait command.

  • Strafe Helper (cg_strafeHelper)

    • Fixed an inaccuracy in strafe helper angle calculations.
    • Since strafe helper now has 11 parameters, we've added a new Strafe Helper Tweaker command (strafe) to easily configure your strafe helper setting. We recommend using this command instead of tweaking the cvar directly. Simply enter strafe to get started.
    • Added two new presets:
      • 3 preset: enables a minimal feature set, most suitable for ingame use with minimal distractions.
      • 4 preset: enables absolutely every feature, most suitable for gaining a better understanding of JKA movement.
    • Added parameters to customize which lines are displayed, and optionally specify a custom color:
      • wawd: display lines for W+A or W+D strafing.
      • ad: display lines for A or D strafing.
      • w: display lines for W strafing.
      • s: display lines for S strafing.
      • sasd: display lines for S+A or S+D strafing.
      • backwardsad: display lines for A or D strafing while moving backwards.
    • Added a new opacity parameter.
    • The crosshair parameter now also accepts a custom color.
    • Added a new tiny option for the height parameter.
    • You can now specify a numerical preset in combination with modifiers, e.g. 2 height=tall.

  • Countdown Timer (nm_timer)

    • Added a new cooldown parameter (useful for preventing accidental double presses).
    • Added a new vstr parameter, which executes the contents of cvar(s) when the timer reaches certain point(s).
    • Added a new hide parameter, which hides the timer from view (most useful in combination with vstr).
    • Fixed a bug where timers having the removeOnDeath parameter sometimes wouldn't work if started moments before spawning.
    • Fixed timers continuing to tick down during server pause.

  • Other improvements

    • Added a new 3 setting to the Unlagged cvar (cg_unlagged) to only disable time nudge while using disruptor.
    • The Carrier Display cvar (cg_drawCarrier) now also displays your allied carrier's location underneath their health and armor.
    • JKChat users are now displayed on the scoreboard with a special icon and are affected by the Hide Spectators cvar (cg_specs) regardless of their name.
    • With Simple Items (cg_simpleItems) enabled, dropped weapons now use a new alternate image, and unavailable weapons (those waiting to respawn) are now slightly more transparent. These changes allow dropped weapons, available spawns, and unavailable spawns to be more visually distinct from each other.
    • Increased the limit for gamma (r_gamma) from 3 to 10.
    • Several other miscellaneous improvements.

  • Other bug fixes

    • Fixed bugs relating to seeing and sniping extremely distant players, particularly in CTF.
    • Fixed weapon spawns sometimes flickering upon pickup.
    • Fixed a bug that could sometimes cause vchat pack loading to fail.
    • Several other miscellaneous bug fixes.
