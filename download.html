Download the latest version of the NewJK executable for Windows (May 17, 2020).

NewJK is our optional improved custom executable that replaces jamp.exe, which launches when you start JKA. It can improve FPS, has many crash fixes and stability improvements, and adds extra capabilities like raw mouse input. While NewJK is not required to use Newmod, we highly recommend it. For more information, see the NewJK page.

If you are also already using Newmod, make sure to update that, too.

To install NewJK manually, put jamp.exe, rd-vanilla_x86.dll, and SDL2.dll in the folder where your existing jamp.exe is located (this is usually the /GameData/ folder). We recommend backing up your existing jamp.exe first.

Release notes for 1.6.0:


  • Unlagged

    • On compatible (base_enhanced) servers, you can now use "unlagged" lag compensation using the new Unlagged cvar (cg_unlagged).

  • Homing Rockets

    • Fixed several base JKA bugs relating to homing rockets.
    • On compatible (base_enhanced) servers, the homing rocket HUD is now properly synchronized with the server, and is now 100% accurate. If you see/hear that you have locked onto a target, you can trust that your rocket is 100% guaranteed to home toward the target.
    • On NON-supporting servers, the homing rocket HUD is now much more accurate (though not 100% accurate).

  • Server Compatibility

    • The scoreboard now supports up to 32 players (increased from 20).
    • NewJK executable only: you can now play on JKA version 1.0.0 servers.
    • NewJK executable only: you can now play on "pure" servers regardless of your mods by using the new Pure Bypass cvar (fs_pureBypass).

  • Calling Votes

    • On compatible (base_enhanced) servers, a new tab has been added to the Call Vote menu, allowing easy access to new votes, e.g. Coin Toss, Reset Flags, etc. This removes the need for dirty serverside vote redirects (e.g. changing "Do Warmup" to do something else).

  • Miscellaneous improvements

    • Potentially improved FPS through some optimizations and fixes.
    • You can now disable meme vchats from spectators while in-game by using the new Spectator Memes cvar (cg_specMemes).
    • When someone calls a vote, the vote text is now displayed in the console, e.g.: "Padawan called a vote: Restart Map (3)"
    • Added translation support for Greek.
    • You now automatically reconnect upon getting kicked by a reliable commands error.
    • Several other miscellaneous improvements.

  • Miscellaneous fixes

    • NewJK executable only: fixed laggy/buggy behavior and "Connection Interrupted" when playing with high latency with a high Max FPS (com_maxFPS).
    • Fixed bugs with typing accented/special letters in non-English keyboards.
    • The Hide Chats cvar (cg_teamChatsOnly) now properly supports hiding chat from CTF racers, and vchat.
    • Fixed a bug relating to the removeOnDeath parameter of the Countdown Timer command (nm_timer).
    • Fixed timers and some other HUD elements not displaying properly while Strafe Helper (cg_strafeHelper) is enabled.
    • Fixed laggy movement on top of other players.
    • Fixed laggy movement near emplaced guns.
    • CTF racers who are lagging no longer appear on the HUD with the Lag Display cvar (cg_drawLag) enabled.
    • CTF racers are no longer affected by the Force Models cvar (cg_forceModel).
    • Several other miscellaneous fixes.
