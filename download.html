Download Newmod 1.3.0 zip file

Download NewJK 1.3.0 zip file
To install Newmod manually, put newmod.pk3 and assetsNewmod.pk3 in your /base/ folder. Remove any conflicting mods, including Smod-related files.
To install NewJK manually, put jamp.exe, rd-vanilla_x86.dll, and SDL2.dll in the folder where your existing jamp.exe is located (this is usually the /GameData/ folder). We recommend backing up your existing jamp.exe first.

Release notes for 1.3.0:

  • cg_forceModel can now be used to force allied and/or enemy team to use multiple models from a pool, instead of just a single model.
  • Added the new cvar cg_strafeHelper, which displays lines indicating optimum angles for strafejumping.
  • Added the new cvar cg_crosshairNameTime, which allows you to configure how long to continue displaying the name of your last crosshair target after they leave your crosshair.
  • Added the new cvar cg_drawForceIcons, which allows you to use an alternate style of force power icons, as well as allowing simple HUD users to use icons instead of text.
  • Added the new cvar cl_shiftToggleConsole, which allows you to choose whether to require holding Shift to open and/or close the console. Depending on your setting, you can now type characters that are normally blocked, such as ` on English keyboard, ^ on German keyboard, etc.
  • Automatic map downloading now works from demo playback.
  • Fixed some more base JKA effects duplication bugs (bowcaster, demp, item spawn cones).
  • Fully removed unnecessary weapon swaying in first-person view.
  • Removed some annoying base JKA warnings that could occasionally popup on screen, such as "Your ghoul2 instance has no model!" or "WARNING: RE_AddPolyToScene: NULL poly shader".
  • Miscellaneous bug fixes and improvements.
