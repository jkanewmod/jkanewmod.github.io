Download the latest version of the NewJK executable for Windows (2023-09-09).

While NewJK is not required to use Newmod, we highly recommend it.

NewJK is our optional improved custom executable that replaces the vanilla JKA executable (jamp.exe). As a custom executable, it is capable of implementing many improvements that are not possible with simple PK3 mods (like Newmod). It can improve FPS, has many crash fixes and stability improvements, and adds extra features like raw mouse input. For more information, see the NewJK page.

If you are also already using Newmod, make sure to update that, too.

To install NewJK manually, put jamp.exe, rd-vanilla_x86.dll, SDL2.dll, steam_api.dll, and steam_appid.txt in the folder where your existing jamp.exe is located (this is usually the /GameData/ folder). We recommend backing up your existing jamp.exe first.

Release notes for 1.11.0:




    CTF position features

    • Added new parameters to the Force Models (cg_forceModel) cvar for forcing models for certain CTF positions.
    • Added new parameters to the Fullbright Models (cg_fullbrightModels) cvar for forcing full brightness for certain CTF positions.
    • Added a new Team Overlay CTF Position Order (cg_teamOverlayCtfPosOrder) cvar, which orders the team overlay by CTF positions.
    • Added a new CTF Position Icons (cg_drawCtfPosIcons) cvar, which displays icons for each CTF position in the scoreboard and/or team overlay.
    • Added a new CTF Position CFGs (cg_ctfPosCFGs) cvar, which automatically executes a CFG depending on your CTF position.

    Auto click

    • Added a new Auto Click (cg_autoClick) cvar, which automatically clicks to respawn when dead.

    Sound volume

    • Added a new Weapon Volume (s_volumeWeapon) cvar, which sets the volume of weapon sounds. You can optionally set different volumes for your own weapon sounds and other players' weapon sounds.
    • Base JKA has bugs that make saber activation/deactivation sounds louder than intended. These bugs are fixed by default with the new Fix Saber Sounds (cg_fixSaberSounds) cvar, although you can optionally disable it if you prefer the loud base JKA saber activation/deactivation sounds.

    TH/TE indicator

    • Added a new TH/TE Indicator (cg_drawThTe) cvar, which displays HUD indicators for team heal and team energize.

    Other fixes and improvements

    • Added a new New Weapon (weapnew) command, which switches to your most recently acquired weapon.
    • Added a new Demo CFGs (cg_demoCFGs) cvar, which lets you execute a CFG depending on whether a demo is playing.
    • Fixed undropped weapons held by corpses (e.g. pistol) continuing to use fullbright.
    • Fixed some base JKA bugs that caused HUD elements to overlap with other HUD elements.
    • Numerous miscellaneous fixes and improvements.
